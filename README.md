# 🎬 Netflix Clone App

This is a Netflix-inspired clone application developed for learning and demonstration purposes.
It is an independent project and is not affiliated with or endorsed by Netflix.

---

## 📌 About the Project

This project is built to practice Android development and understand how modern OTT platforms work, focusing on UI design and user experience.

---

## 🚀 Features

* 👤 User Selection Screen (Who’s watching / Kids profile)
* 🏠 Home Section (Browse content - TV Shows, Movies, My List)
* 🔍 Search Functionality
* 👤 Profile Section
